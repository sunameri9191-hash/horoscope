# 🔮 ホロスコープ作成サイト

**URL:** https://sunameri9191-hash.github.io/horoscope/

フロントエンドのみ(HTML/CSS/JavaScript)で動作するブラウザ対応のホロスコープ作成Webサイトです。サーバーは不要で、GitHub Pages で無料公開しています。

---

## 📁 ファイル構成

```
horoscope/
├── index.html        # ページ構造 (HTML)
├── style.css         # デザイン・レイアウト
├── app.js            # アプリの全ロジック
├── astro.bundle.js   # 天体計算ライブラリ(変更不要)
├── README.md         # このファイル
└── SETUP.md          # VSCode・Git セットアップ手順
```

> ⚠️ `astro.bundle.js` は `astronomy-engine` + `circular-natal-horoscope-js` をバンドルしたファイルです。基本的に変更不要です。

## ✨ 機能一覧

### ホロスコープ
| タイプ | 内容 |
|------|------|
| ネイタル | 出生時の天体配置チャート |
| 二重円 | ネイタル + トランジット(指定日時) |
| 三重円 | ネイタル + プログレス + トランジット |
| 一日一年法 | セカンダリープログレッション (☽月 / ☉太陽) |

### その他
- **天体暦** … 日ごとの全天体の度数・サインを一覧表示(月めくり式)
- **🖨 印刷/PDF** … ホロスコープホイール + プロフィール + 詳細データをA4で出力
- **天体暦印刷** … 期間を選んでA4縦・2ヶ月/ページで出力
- **満月・新月チェック** … カレンダー直近の満月・新月をスクロール表示

### プロフィール
- 名前・生年月日・出生時刻(不明時は正午で仮算出)・出生地(緯度経度)・ハウスシステムを設定
- データはブラウザの `localStorage` に保存(各端末ごとに独立)
- 複数プロフィールの切り替え可能

### ハウスシステム
プラシダス / コッホ / イコール / ホールサイン / カンパナス / レジオモンタナス / ポルフィリオティック

### サインの色分け(四元素)
| 元素 | サイン | 色 |
|------|--------|-----|
| 🔥 火 | 牡羊・獅子・射手 | 赤 |
| 🌿 地 | 牡牛・乙女・山羊 | オレンジ |
| 💨 風 | 双子・天秤・水瓶 | 緑 |
| 💧 水 | 蟹・蠍・魚 | 青 |

## 🔧 使用ライブラリ

- [astronomy-engine](https://github.com/cosinekitty/astronomy) … NASAのJPLデータ準拠の天体計算
- [circular-natal-horoscope-js](https://github.com/0xStarcat/CircularNatalHoroscopeJS) … ホロスコープ・ハウス計算

計算方式: **トロピカル黄道帯**。天体暦の時刻は **日本標準時(JST)** 基準

## 📝 変更するときのポイント

### `index.html`
- HTMLの構造とバージョン番号(`build vXX`)を管理
- `app.js` の読み込み行に `?v=XX` のキャッシュバスティングを付けている
- **変更後は必ずバージョン番号を上げる** (`?v=28` → `?v=29`)

### `style.css`
- CSS変数(`:root {}`)でサイト全体の色・背景を一括管理
- 印刷スタイルは `app.js` 内の `openPrintWindow()` 関数に書いてある

### `app.js`
- 先頭の定数 `SIGNS` / `BODIES` / `ASPECTS` / `RING_COLORS` を変えると見た目に反映される
- 各機能は `init〇〇()` という関数で初期化されている
- 最終行付近の `init〇〇()` 呼び出しが全機能の起動ポイント
